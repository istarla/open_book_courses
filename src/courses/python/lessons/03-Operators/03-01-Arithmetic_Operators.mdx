# What are Operators?

<section id="What-are-operators?"></section>
Operators are special symbols or keywords in Python that perform operations on values. These values are called operands. For example, in the expression `5 + 3`, the `+` is the operator, and `5` and `3` are the operands.

Today, we focus on arithmetic operators, which are used to perform mathematical calculations. These work primarily with numeric types like `int` and `float`.

## Detailed Explanations

<section id="Detailed-explanations"></section>

### Addition, Subtraction, Multiplication
These behave as you'd expect from standard mathematics.

```python
x = 15
y = 4

add_result = x + y
sub_result = x - y
mul_result = x * y

print(f"{x} + {y} = {add_result}")   # Output: 15 + 4 = 19
print(f"{x} - {y} = {sub_result}")   # Output: 15 - 4 = 11
print(f"{x} * {y} = {mul_result}")   # Output: 15 * 4 = 60
```

### Division (/)
Standard division always produces a float, even if the result is a whole number.

```python
num1 = 20
num2 = 5
num3 = 7

div_result1 = num1 / num2 # Result is a whole number, but still float
div_result2 = num1 / num3 # Result is fractional

print(f"{num1} / {num2} = {div_result1}", type(div_result1)) # Output: 20 / 5 = 4.0 <class 'float'>
print(f"{num1} / {num3} = {div_result2}", type(div_result2)) # Output:
```
### Floor Division (//)

Floor division performs the division but discards the fractional part, rounding down to the nearest integer. If operands are integers, the result is an integer. If either operand is a float, the result is a float (but still rounded down).

```python
a = 17
b = 4
c = -17

floor_div1 = a // b
floor_div2 = c // b # Note behavior with negative numbers
floor_div3 = 17.0 // 4 # With a float operand

print(f"{a} // {b} = {floor_div1}", type(floor_div1)) # Output: 17 // 4 = 4 <class 'int'>
print(f"{c} // {b} = {floor_div2}", type(floor_div2)) # Output: -17 // 4 = -5 <class 'int'> (rounds down towards negative infinity)
print(f"17.0 // 4 = {floor_div3}", type(floor_div3)) # Output: 17.0 // 4 = 4.0 <class 'float'>
```

### Modulus (%)

The modulus operator gives you the remainder after integer division. It's very useful for tasks like checking if a number is even or odd, or cycling through values.

```python
num = 14
divisor = 5

remainder = num % divisor
print(f"The remainder of {num} divided by {divisor} is {remainder}") # Output: 4

# Check if a number is even or odd
check_num = 21
if check_num % 2 == 0:
    print(f"{check_num} is even")
else:
    print(f"{check_num} is odd") # Output: 21 is odd
```

### Exponentiation (**)

Raises the first number to the power of the second.

```python
base = 2
exponent = 5

power_result = base ** exponent
print(f"{base} to the power of {exponent} is {power_result}") # Output: 2 to the power of 5 is 32

# Can use floats as well
root_result = 16 ** 0.5 # Square root
print(f"The square root of 16 is {root_result}") # Output: 4.0
```

### Operator Precedence

<section id="Operator-Precedence"></section>

When an expression has multiple operators, Python follows rules of precedence to determine the order of evaluation (similar to PEMDAS/BODMAS in mathematics).

- ** (Exponentiation) - Highest precedence
- *, /, //, % (Multiplication, Division, Floor Division, Modulus) - Evaluated left-to-right
- +, - (Addition, Subtraction) - Lowest precedence, evaluated left-to-right

You can use parentheses `()` to override the default precedence and force parts of an expression to be evaluated first.

```python
result1 = 5 + 2 * 3       # Multiplication first: 5 + 6 = 11
result2 = (5 + 2) * 3     # Parentheses first: 7 * 3 = 21
result3 = 10 / 2 * 3      # Left-to-right for / and *: (10/2)*3 = 5.0*3 = 15.0
result4 = 2 ** 3 ** 2     # Exponentiation right-to-left: 2**(3**2) = 2**9 = 512
result5 = (2 ** 3) ** 2   # Using parentheses: (8)**2 = 64

print(f"5 + 2 * 3 = {result1}")
print(f"(5 + 2) * 3 = {result2}")
print(f"10 / 2 * 3 = {result3}")
print(f"2 ** 3 ** 2 = {result4}")
print(f"(2 ** 3) ** 2 = {result5}")
```
### Example 1: Calculating Area and Circumference
```python
import math  # Import the math module for pi value

radius = 7.5

# Calculate area: pi * r^2
area = math.pi * (radius ** 2)

# Calculate circumference: 2 * pi * r
circumference = 2 * math.pi * radius

print(f"Radius: {radius}")
print(f"Area: {area:.2f}")  # Format to 2 decimal places
print(f"Circumference: {circumference:.2f}")
```
