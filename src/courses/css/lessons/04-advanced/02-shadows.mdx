<section id="text-shadows">
## Text Shadows

Add depth to text with shadow effects.

```css
.text-shadow {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
}
.multiple-shadows {
  text-shadow: 1px 1px 2px red, 0 0 1em blue;
}
```
</section>

<section id="box-shadows">
## Box Shadows

Create shadow effects for elements.

```css
.box-shadow {
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
.inset-shadow {
  box-shadow: inset 0 2px 4px rgba(0, 0, 0, 0.2);
}
```
</section>
