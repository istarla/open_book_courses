<section id="table-borders">
## Table Borders

Add borders to table elements for better structure.

```css
table { border-collapse: collapse; }
td, th {
  border: 1px solid #ddd;
  padding: 8px;
}
```
</section>

<section id="table-spacing">
## Cell Spacing & Padding

Control spacing within table cells.

```css
table {
  border-spacing: 10px;
}
td {
  padding: 12px 15px;
}
```
</section>

<section id="table-styling">
## Table Styling

Enhance table appearance with styling.

```css
tr:nth-child(even) {
  background-color: #f2f2f2;
}
th {
  background-color: #4CAF50;
  color: white;
}
```
</section>
