    <h1 class="day-content-h1-tags">CSS Comments and Basic Color Properties</h1><br>
    <div class="content-container p-6 md:p-10">
      <p class="mb-4">
        On our third day, we'll cover how to add comments to your CSS code for
        better readability and maintainability. We'll also dive into one of the
        most fundamental aspects of styling: CSS color properties, focusing on
        how to define colors for text and backgrounds.
      </p>

      <h2 class="text-2xl font-semibold section-title">CSS Comments</h2>
      <p>
        Comments in CSS are used to explain your code, make it more readable, or
        temporarily disable parts of your stylesheet. Browsers ignore comments,
        so they don't affect the functionality of your CSS.
      </p>
      <p class="mt-2">
        CSS comments start with <code>/*</code> and end with <code>*/</code>.
        They can span multiple lines.
      </p>

      <p><strong>Syntax:</strong></p>
      <pre
        class="code-block rounded-md"
      ><code class="language-css line-numbers">/* This is a single-line comment */

p {
  color: green; /* Set paragraph text color to green */
  font-size: 16px;
}

/*
This is a
multi-line comment.
It can be used to explain a larger block of code
or to temporarily comment out a section.

.unused-style {
  background-color: red;
  border: 1px solid black;
}
*/

h1 {
  /* color: blue; Temporarily disabled this rule */
  font-weight: bold;
}
</code></pre>

      <h3 class="text-xl font-medium mt-4 mb-2">Why Use Comments?</h3>
      <ul class="list-disc list-inside ml-4 my-2 space-y-1">
        <li>
          <strong>Explanation:</strong> Clarify complex or non-obvious CSS rules
          for yourself or other developers.
        </li>
        <li>
          <strong>Organization:</strong> Divide your stylesheet into logical
          sections.
        </li>
        <li>
          <strong>Debugging:</strong> Temporarily disable rules to isolate
          issues.
        </li>
        <li>
          <strong>Documentation:</strong> Provide information about authorship,
          version, or purpose of the styles.
        </li>
      </ul>
      <div class="note">
        <p>
          <strong>Tip:</strong> Write comments that explain <em>why</em> you did
          something, not just <em>what</em> you did. Good code often explains
          itself, but the reasoning behind a choice might not be obvious.
        </p>
      </div>

      <h2 class="text-2xl font-semibold section-title">
        Basic CSS Color Properties
      </h2>
      <p>
        Color is a fundamental aspect of web design. CSS provides several
        properties to control the colors of various parts of your HTML elements.
        Today, we'll focus on two primary ones: <code>color</code> for text and
        <code>background-color</code> for element backgrounds.
      </p>
      <p>
        We'll explore different ways to define color values (like keywords, hex,
        RGB) in a later lesson. For now, we'll primarily use named colors and
        simple hex codes.
      </p>

      <h3 class="text-xl font-medium mt-4 mb-2">
        1. The <code>color</code> Property
      </h3>
      <p>
        The <code>color</code> property is used to set the color of the text
        content of an HTML element.
      </p>
      <p><strong>Example:</strong></p>
      <pre
        class="code-block rounded-md"
      ><code class="language-css line-numbers">/* styles.css */
h1 {
  color: navy; /* Sets the text color of h1 elements to navy */
}

p {
  color: #333333; /* Sets paragraph text color to a dark gray */
}

.special-text {
  color: crimson; /* Sets text color for elements with class 'special-text' */
}
</code></pre>
      <pre
        class="code-block rounded-md"
      ><code class="language-html line-numbers">&lt;!-- index.html --&gt;
&lt;h1&gt;Main Title&lt;/h1&gt;
&lt;p&gt;This is a standard paragraph with dark gray text.&lt;/p&gt;
&lt;p class="special-text"&gt;This paragraph has special crimson text.&lt;/p&gt; </code></pre>
      <p>
        In this example, <code>&lt;h1&gt;</code> elements will have navy blue
        text, regular paragraphs will have dark gray text, and any element with
        the class <code>special-text</code> will have crimson text.
      </p>

      <h3 class="text-xl font-medium mt-4 mb-2">
        2. The <code>background-color</code> Property
      </h3>
      <p>
        The <code>background-color</code> property sets the background color of
        an element. The background extends to the outer edge of the element's
        padding (or border if <code>box-sizing: border-box</code> is used and
        padding is included).
      </p>
      <p><strong>Example:</strong></p>
      <pre
        class="code-block rounded-md"
      ><code class="language-css line-numbers">/* styles.css */
body {
  background-color: #f0f8ff; /* AliceBlue background for the whole page */
}

div.highlight {
  background-color: yellow;
  padding: 10px; /* Padding to see the background color better */
}

button {
  background-color: seagreen;
  color: white; /* Text color on the button */
  padding: 8px 15px;
  border: none;
} </code></pre>
      <pre
        class="code-block rounded-md"
      ><code class="language-html line-numbers">&lt;!-- index.html --&gt;
&lt;body&gt;
  &lt;div class="highlight"&gt;
    This div has a yellow background.
  &lt;/div&gt;
  &lt;p&gt;This paragraph is on the page's AliceBlue background.&lt;/p&gt;
  &lt;button&gt;Click Me&lt;/button&gt;
&lt;/body&gt;</code></pre>
      <p>
        Here, the entire page background is set to a light blue (AliceBlue). The
        <code>div</code> with class <code>highlight</code> has a yellow
        background. The button has a seagreen background and white text for
        contrast.
      </p>

      <h3 class="text-xl font-medium mt-4 mb-2">Named Colors</h3>
      <p>
        CSS supports a list of predefined
        <a
          href="https://www.w3schools.com/colors/colors_names.asp"
          target="_blank"
          class="text-blue-600 hover:underline"
          >named colors</a
        >
        (e.g., <code>red</code>, <code>blue</code>, <code>green</code>,
        <code>lightcoral</code>, <code>mediumseagreen</code>). These are easy to
        remember and use for basic styling or quick prototyping.
      </p>
      <p><strong>Example using named colors:</strong></p>
      <div class="flex items-center mb-2">
        <div class="color-box" style="background-color: tomato"></div>
        <span>Tomato</span>
      </div>
      <div class="flex items-center mb-2">
        <div class="color-box" style="background-color: dodgerblue"></div>
        <span>DodgerBlue</span>
      </div>
      <div class="flex items-center mb-4">
        <div class="color-box" style="background-color: gold"></div>
        <span>Gold</span>
      </div>
      <pre
        class="code-block rounded-md"
      ><code class="language-css line-numbers">.error-message {
  color: red;
  background-color: lightpink;
}

.success-message {
  color: green;
  background-color: lightgreen;
}

nav {
  background-color: steelblue;
  padding: 1em;
}

nav a {
  color: white;
} </code></pre>
      <pre
        class="code-block rounded-md"
      ><code class="language-html line-numbers">&lt;!-- index.html --&gt;
&lt;nav&gt;
  &lt;a href="#"&gt;Home&lt;/a&gt; | &lt;a href="#"&gt;About&lt;/a&gt;
&lt;/nav&gt;
&lt;p class="error-message"&gt;Something went wrong!&lt;/p&gt;
&lt;p class="success-message"&gt;Operation successful!&lt;/p&gt; </code></pre>
      <p>
        While named colors are convenient, for more precise color control and a
        wider range of colors, you'll often use hexadecimal, RGB, or HSL values,
        which we will cover in detail in Week 5.
      </p>

        <h2>Quiz Time!</h2>
        <div>
        <p class="question">
          1. How do you write a single-line comment in CSS?
        </p>
        <p class="answer"><code>/* This is a comment */</code></p>
      </div>
      <div>
        <p class="question">
          2. Can CSS comments span multiple lines? If so, how?
        </p>
        <p class="answer">
          Yes, they can. They start with <code>/*</code> and end with
          <code>*/</code>, and anything between these markers is a comment,
          regardless of the number of lines.
        </p>
      </div>
      <div>
        <p class="question">
          3. Which CSS property is used to change the text color of an element?
        </p>
        <p class="answer">The <code>color</code> property.</p>
      </div>
      <div>
        <p class="question">
          4. Which CSS property is used to change the background color of an
          element?
        </p>
        <p class="answer">The <code>background-color</code> property.</p>
      </div>
      <div>
        <p class="question">
          5. Write a CSS rule to make all <code>&lt;h2&gt;</code> elements have
          blue text and a lightgray background.
        </p>
        <p class="answer">Answer</p>
        <pre class="code-block line-numbers"><code class="language-css line-numbers">h2 {
            color: blue;
            background-color: lightgray;
          }  </code></pre>
