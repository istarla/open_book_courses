## What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents.

```css
/* Basic CSS syntax */
selector {
  property: value;
}
```

CSS follows a simple syntax structure with selectors, properties, and values.

```css
/* CSS rule structure */
```

<section id="how-to-use">
## Ways to Add CSS

There are three main ways to add CSS to HTML: inline, internal, and external.

```html
<!-- Inline CSS -->
<p style="color: red;">This is red text</p>

<!-- Internal CSS -->
<style>
  p { color: blue; }
</style>

<!-- External CSS -->
<link rel="stylesheet" href="styles.css">
```
</section>
