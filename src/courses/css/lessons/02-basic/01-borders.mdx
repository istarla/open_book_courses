<section id="border-styles">
## Border Styles

CSS provides various border styles for elements.

```css
.border-solid { border-style: solid; }
.border-dashed { border-style: dashed; }
.border-dotted { border-style: dotted; }
```
</section>

<section id="border-width">
## Border Width

Control the thickness of borders using border-width property.

```css
.thin-border { border-width: 1px; }
.thick-border { border-width: 5px; }
.mixed-border { border-width: 1px 2px 3px 4px; }
```
</section>

<section id="border-color">
## Border Color

Set border colors using various color formats.

```css
.red-border { border-color: red; }
.hex-border { border-color: #3498db; }
.rgba-border { border-color: rgba(255, 0, 0, 0.5); }
```
</section>
