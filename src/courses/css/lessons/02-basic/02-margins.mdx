## Margin Properties

Margins create space around elements outside their borders.

```css
.margin-top { margin-top: 10px; }
.margin-right { margin-right: 15px; }
.margin-bottom { margin-bottom: 20px; }
.margin-left { margin-left: 25px; }
```
</section>

<section id="margin-shorthand">
## Shorthand Margin Syntax

Use shorthand properties for efficient margin declarations.

```css
.margin-all { margin: 10px; }
.margin-vertical { margin: 10px 20px; }
.margin-mixed { margin: 10px 15px 20px 25px; }
```
</section>
